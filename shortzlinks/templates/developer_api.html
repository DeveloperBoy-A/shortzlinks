API Endpoints

Get Account Stats

GET /api/dev/stats Headers: X-API-Key: your_api_key

Response:

{ "username": "your_username", "total_links": 10, "total_clicks": 1000, "total_earnings": 5.50, "balance": 2.50 }

Create Shortened Link

POST /api/dev/create Headers: X-API-Key: your_api_key Content-Type: application/json { "url": "https://example.com/very/long/url" }

Response:

{ "success": true, "short_code": "abc123", "short_url": "https://yoursite.com/ad/abc123", "original_url": "https://example.com/very/long/url" }

Get All Links

GET /api/dev/links Headers: X-API-Key: your_api_key

Response:

{ "total": 10, "links": [ { "short_code": "abc123", "original_url": "https://example.com", "clicks": 100, "earnings": 0.50, "created_at": "2024-01-15T10:30:00" } ] }

Get CPM Rates (Public)

GET /api/dev/rates (No authentication required)

Response:

{ "total_countries": 195, "rates": [ { "country_code": "US", "country_name": "United States", "cpm": 8.50, "network": "adsterra", "updated_at": "2024-01-15T00:00:00" } ] }

Code Examples

JavaScript/Node.js

const apiKey = 'your_api_key'; // Create a short link fetch('https://yourdomain.com/api/dev/create', { method: 'POST', headers: { 'X-API-Key': apiKey, 'Content-Type': 'application/json' }, body: JSON.stringify({ url: 'https://example.com/long/url' }) }) .then(res => res.json()) .then(data => console.log(data.short_url));

Python

import requests api_key = 'your_api_key' headers = {'X-API-Key': api_key} # Get stats response = requests.get( 'https://yourdomain.com/api/dev/stats', headers=headers ) print(response.json()) # Create short link response = requests.post( 'https://yourdomain.com/api/dev/create', headers=headers, json={'url': 'https://example.com/long/url'} ) print(response.json()['short_url'])

cURL

# Get account stats curl -X GET https://yourdomain.com/api/dev/stats \ -H "X-API-Key: your_api_key" # Create short link curl -X POST https://yourdomain.com/api/dev/create \ -H "X-API-Key: your_api_key" \ -H "Content-Type: application/json" \ -d '{"url":"https://example.com/long/url"}'

Rate Limits

  • 100 requests per minute for authenticated endpoints
  • 1000 requests per hour per API key
  • No limit on public endpoints (rates)

Error Handling
